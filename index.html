1. Selector dan Properti

Selector adalah elemen yang kita pilih untuk diterapkan gaya. Setiap selector menentukan elemen HTML yang akan dipengaruhi oleh aturan CSS. Properti adalah atribut gaya yang diterapkan pada elemen yang dipilih oleh selector.

Contoh:


p {
  color: red;
}

Ini adalah paragraf yang akan terlihat berwarna merah.

2. Jenis-Jenis Selector

Berbagai jenis selector digunakan untuk memilih elemen HTML yang spesifik untuk diterapkan gaya:

  • Selektor Tag: Memilih elemen berdasarkan nama tag HTML.
  • Selektor Kelas: Memilih elemen dengan kelas tertentu, diawali dengan titik.
  • Selektor ID: Memilih elemen dengan ID tertentu, diawali dengan pagar.
  • Selektor Atribut: Memilih elemen berdasarkan atribut tertentu.
  • Selektor Descendant: Memilih elemen yang merupakan keturunan dari elemen lain.
  • Selektor Child: Memilih elemen yang merupakan anak langsung dari elemen lain.

Contoh:


/* Selektor Tag */
p {
  color: blue;
}

/* Selektor Kelas */
.box {
  border: 1px solid black;
}

/* Selektor ID */
#unique {
  background-color: yellow;
}

/* Selektor Atribut */
input[type="text"] {
  border: 2px solid green;
}

/* Selektor Descendant */
div p {
  color: gray;
}

/* Selektor Child */
ul > li {
  list-style-type: square;
}

3. Box Model

CSS Box Model adalah konsep yang menjelaskan bagaimana elemen HTML diatur dan ditata. Ini terdiri dari Content, Padding, Border, dan Margin.

Contoh:


div {
  width: 200px; /* Content width */
  padding: 10px; /* Space inside element */
  border: 1px solid black; /* Border around padding */
  margin: 20px; /* Space outside element */
}
Ini adalah contoh elemen dengan box model CSS.

4. Layout yang Unik

Flexbox: Metode untuk menyusun elemen dalam satu dimensi (baris atau kolom) dengan cara yang fleksibel.


.container {
  display: flex;
  justify-content: space-between; /* Align items horizontally */
}

Grid: Metode untuk menyusun elemen dalam dua dimensi (baris dan kolom) dengan cara yang lebih kompleks.


.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */
  gap: 10px;
}

5. Responsive Design

Desain responsif memungkinkan situs web menyesuaikan tampilan berdasarkan ukuran layar perangkat. Ini biasanya dilakukan dengan menggunakan media queries.


@media (max-width: 600px) {
  .container {
    flex-direction: column; /* Stack items vertically on small screens */
  }
}

6. Animasi

CSS Animasi memungkinkan pembuatan animasi dengan menggunakan @keyframes dan properti animation. Animasi dapat mengubah properti CSS dari satu nilai ke nilai lainnya secara bertahap.

Contoh:


/* Keyframe Animations */
@keyframes colorChange {
    0% { background-color: red; }
    50% { background-color: green; }
    100% { background-color: blue; }
}

@keyframes moveAndScale {
    0% {
        transform: translateX(0) scale(1);
        background-color: red;
    }
    50% {
        transform: translateX(100px) scale(1.5);
        background-color: yellow;
    }
    100% {
        transform: translateX(0) scale(1);
        background-color: blue;
    }
}

.animated-box {
    width: 100px;
    height: 100px;
    animation: colorChange 4s infinite, moveAndScale 4s infinite;
}

7. Variabel

CSS Variabel memungkinkan Anda menyimpan nilai yang dapat digunakan di beberapa tempat dalam stylesheet.

Contoh:


:root {
  --main-color: #3498db;
  --secondary-color: #2ecc71;
}

.container {
  color: var(--main-color);
  background-color: var(--secondary-color);
}
Ini adalah contoh menggunakan variabel CSS.

8. Custom Property

Custom Property adalah cara untuk mendefinisikan variabel dalam CSS yang dapat digunakan di seluruh stylesheet.

Contoh:


:root {
  --primary-color: #2980b9;
}

.button {
  background-color: var(--primary-color);
  color: #fff;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}
